# 🌟 Quotron

**Quotron** is a modern web app to generate, explore, and discover inspiring quotes with a clean and aesthetic UI.

---

## 🚀 Features

* 🎲 Random quote generator
* 🔍 Search, filter & sort quotes
* 🌅 Daily Quote (changes every day)
* 🌙 Dark / Light mode
* 📋 Copy quotes instantly

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Fetch API
* LocalStorage

---

## 🔗 API

* https://dummyjson.com/quotes

---

## 📂 Pages

* Home
* Explore
* Daily

---

## ▶️ Run

Open `home.html` with Live Server or browser.
