# 🌌 FanVerse – Explore the Merch Universe

## 📌 Project Overview

FanVerse is a gamified web application that helps users discover trusted websites for fandom-based merchandise. Instead of a traditional interface, the application presents different fandom categories as interactive “planets” in a space-themed environment.

Users can explore categories like Anime, Music, Sports, and Marvel/DC, and find reliable platforms to purchase merchandise related to their interests.

---

## 🎯 Purpose

Finding authentic and reliable merch for different fandoms can be difficult, as users often need to search across multiple websites. FanVerse solves this problem by acting as a curated gateway to trusted merch platforms, all in one place.

---

## 🌐 API Used

This project uses the Fake Store API to fetch and display trending product data dynamically.

API Link: https://fakestoreapi.com/

## ✨ Features

* 🌌 Gamified space-themed interface

* 🔍 Search functionality
